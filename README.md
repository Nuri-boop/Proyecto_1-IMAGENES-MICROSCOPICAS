# Proyecto_1-IMAGENES-MICROSCOPICAS
Detección, segmentación y conteo automatizado de células en imágenes microscópicas.
Automated detection, segmentation and counting of cells in microscopic images.


1.- Requerimientos
  Esta libreria hace uso de las siguientes librerias por lo que debes asegurarte de tenerlas antes de intentar usarla.
  numpy
  matplotlib
  opencv-python
  scipy
